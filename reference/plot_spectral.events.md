# Plot Spectral Events

Plot the results of
[reference.group.spectral.events](https://nlaniewski.github.io/flowstate/reference/reference.group.spectral.events.md);
visual assessment of selected 'spectral events' is used to gauge
function performance.

## Usage

``` r
plot_spectral.events(flowstate.object.reference)
```

## Arguments

- flowstate.object.reference:

  A `flowstate` object as returned from
  [reference.group.spectral.events](https://nlaniewski.github.io/flowstate/reference/reference.group.spectral.events.md).

## Value

A list of [ggplot](https://ggplot2.tidyverse.org/reference/ggplot.html)
objects.
