# Plot Spectral Traces

Plots a 'spectral trace' – normalized `[0,1]` emission for all
detectors.

## Usage

``` r
plot_spectral.trace(ref.medians)
```

## Arguments

- ref.medians:

  The return of
  [reference.group.medians](https://nlaniewski.github.io/flowstate/reference/reference.group.medians.md).

## Value

A list of [ggplot](https://ggplot2.tidyverse.org/reference/ggplot.html)
objects.
